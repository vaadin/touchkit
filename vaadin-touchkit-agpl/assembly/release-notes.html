Overview

Vaadin TouchKit adds components, utilities and a theme tailored for touch devices for use with Vaadin Framework 7.

Package Contents

Inside the ZIP installation package you will find:

  • TouchKit JARs
  • The TouchKit manual (a chapter of Book of Vaadin)
  • README and licensing information

See the README.html in the installation package for more information about the package contents. The bundled manual gives more detailed instructions.

New in Vaadin TouchKit ${Implementation-Version}

The following features and enhancements are new in Vaadin TouchKit ${Implementation-Version}:

  • Vaadin 7 support
  • Easier creation of custom themes
  • UrlField - A HTML5 URL field
  • DatePicker - A HTML5 field for native picking of dates and times with a fallback mode for when not supported
  • Support for HTML5 local storage from the server side
  • Application cache handling reports when there is a new version available

Change Log

For a complete list of changes in this release, please see the list of closed tickets.

Known Issues

  • Android 2.3 support is limited due to the buggy browser on these devices.
