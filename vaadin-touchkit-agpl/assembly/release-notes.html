Overview

Vaadin TouchKit adds components, utilities and a theme tailored for touch devices for use with Vaadin Framework 7.

Package Contents

Inside the ZIP installation package you will find:

  • TouchKit JARs
  • README and licensing information

See the README.html in the installation package for more information about the package contents. The bundled manual gives more detailed instructions.
